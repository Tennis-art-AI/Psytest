# PsyTest — Клиническая самодиагностика психических расстройств

Статический сайт на GitHub Pages. Нет бэкенда, нет базы данных, нет аналитики.  
Все вычисления — в браузере пользователя.

## Живой сайт

После деплоя: `https://YOUR_USERNAME.github.io/psytest/`

## Что внутри

| Файл | Описание |
|------|----------|
| `index.html` | Главная страница — методология, 10 вкладок расстройств |
| `depression.html` | Диагностика депрессивного расстройства (БДР) — 26 вопросов, 5 доменов |

## Как запустить локально

```bash
# Клонировать репозиторий
git clone https://github.com/YOUR_USERNAME/psytest.git
cd psytest

# Запустить локальный сервер (Python 3)
python3 -m http.server 8000

# Открыть в браузере
# http://localhost:8000
```

Или просто открыть `index.html` в браузере — сайт работает полностью офлайн.

## Как задеплоить на GitHub Pages

### 1. Создать репозиторий

```bash
git init
git add .
git commit -m "Initial commit: PsyTest v1"
git remote add origin https://github.com/YOUR_USERNAME/psytest.git
git branch -M main
git push -u origin main
```

### 2. Включить GitHub Pages с Actions

В репозитории: **Settings → Pages → Source → GitHub Actions**

После первого push CI/CD запустится автоматически.  
Сайт будет доступен через ~2 минуты по адресу:  
`https://YOUR_USERNAME.github.io/psytest/`

### 3. Все последующие обновления

```bash
git add .
git commit -m "Описание изменений"
git push
```

GitHub Actions автоматически пересоберёт и задеплоит.

## Методология

Каждая диагностика объединяет 4 уровня инструментов:

1. **Скрининговые** (PHQ-9, GAD-7, PC-PTSD-5) — быстрая проверка «есть ли проблема»
2. **Оценка тяжести** (HAM-D, MADRS, BDI-II, CAPS-5) — доменная структура
3. **Специализированные конструкты** (PSWQ, IUS-12, MCQ-30, RRS) — механизмы
4. **Теоретические модели** (Beck, Dugas, Wells, Nolen-Hoeksema) — хронификация

**Уникальные особенности:**
- Дифференциальные веса согласно DSM-5 иерархии (0–5 для ядерных, 0–3 для второстепенных)
- Три режима оценки: последние 2 недели / за всю жизнь / худший период
- Учёт «плавающих» (волнообразных) симптомов
- Доменное профилирование (5 доменов, не только общий балл)
- История эпизодов → тактика лечения в выводах
